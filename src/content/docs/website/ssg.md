## 使用程序创建内容网站

### 网站和博客

- Astro

https://docs.astro.build/zh-cn/getting-started/

Astro 是一个新型的 Web 框架，适合构建以内容驱动的网站。它减少了 JavaScript 的开销和复杂性，加载速度快，具有良好的 SEO 特性。

- Hugo

https://gohugo.io/documentation/

https://hugo.opendocs.io/

网站构建框架。

- Hexo

https://hexo.io/zh-cn/docs/

博客和网站构建程序。
