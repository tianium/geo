- IDE

    一个代码编辑器，推荐VSCode。

    https://code.visualstudio.com/

    - 也可以使用Zed，一个Rust开发的轻量级代码编辑器。

      https://zed.dev/

- GitHub
    
    最流行的代码托管平台。

    https://github.com/

    - 另一个由非营利组织运行的隐私友好型代码托管平台。

      https://codeberg.org/
